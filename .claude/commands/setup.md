# 初回セットアップ

環境の確認と、現在の進捗状況を把握する。
すべての応答を日本語で行うこと。

## Steps

### Step 1: 環境確認
以下をコマンドで確認する：
- `git --version` → Git がインストールされているか
- このフォルダが git リポジトリになっているか（`git status`）

結果をわかりやすく報告する。不足があればインストール手順を案内する。

### Step 2: リポジトリの状態確認
- 発注者.md を読み、どのセクションが記入済みかチェック
- products/ フォルダに商品ルールファイルがいくつあるか確認（_template.md は除く）
- gas/Config.gs を読み、設定が記入済みかチェック

### Step 3: 次のステップを案内

以下のフォーマットで出力する：

```
✅ 環境確認完了

📋 現在の進捗：
- 発注者.md: [未記入 / 記入途中 / 完成]
- 商品ルール: [N件作成済み / 未作成]
- GAS設定: [未設定 / 設定済み]
- GAS実装: [未着手 / Phase X まで完了]

👉 次のステップ:
  [状況に応じた案内]
  - 発注者.mdが空 → 「/build-knowledge でナレッジの言語化から始めましょう」
  - 発注者.mdが途中 → 「/build-knowledge で続きから再開できます」
  - 発注者.mdが完成 → 「/dev でGASの実装に進めます」
```
