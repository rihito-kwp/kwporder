# GAS（Google Apps Script）の使い方

このフォルダのファイルは、Google Apps Script エディタにコピペして使います。

---

## GASエディタの開き方

1. https://script.google.com にアクセス
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「KW発注自動化」に変更

## ファイルの貼り付け方

1. デフォルトの `コード.gs` を開いて中身を全部消す
2. このフォルダの `Code.gs` の中身をコピペ → 保存
3. 左の「+」→「スクリプト」→ ファイル名に `Config` と入力
4. `Config.gs` の中身をコピペ → 保存

## テスト実行

1. Code.gs の上部ドロップダウンから実行したい関数を選ぶ
2. 「▶ 実行」ボタンを押す
3. 初回は「権限を確認」→ Googleアカウントで許可

## 自動実行（トリガー）の設定

1. 左メニューの「トリガー」（時計アイコン）
2. 右下「トリガーを追加」
3. 関数: `checkInventory`
4. イベントソース: 「時間主導型」
5. タイプ: 「日タイマー」
6. 時刻: 好きな時間（例: 午前9〜10時）
