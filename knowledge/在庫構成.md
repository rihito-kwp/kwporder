# チャネル別の在庫構成

## 在庫の保管場所と役割

| 保管場所 | 役割・用途 | 備考 |
|---------|-----------|------|
| 国立倉庫（クニロジ） | Shopify注文・卸注文の発送 | |
| FBA倉庫 | Amazon販売専用 | Amazon FBA |
| 中国倉庫（ヲヲフェニックス） | 全製品の製造後ストック | 発注元・出荷元 |

## チャネル間の在庫の流れ

- 中国倉庫（ヲヲフェニックス）→ 国立倉庫 → Shopify注文・卸注文
- 中国倉庫（ヲヲフェニックス）→ FBA倉庫 → Amazon

## 売上データの管理

| チャネル | データ取得方法 | 更新頻度 |
|---------|-------------|---------|
| Shopify | Shopify API連携スプレッドシート（30日間売上） | 毎日朝5時に自動更新 |
| 卸 | 同スプレッドシート内のセル | リアルタイム |
| Amazon | 在庫レポートを手動DL | 手動更新 |
