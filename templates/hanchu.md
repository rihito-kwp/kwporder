# ヲヲフェニックス向け 発注依頼テンプレート

> 普段送っている依頼文をベースに、自動生成用に整えたもの。
> `/build-knowledge` の「ヲヲフェニックスとのやり取り」セクションで内容を確定させる。

---

## テンプレート

```
[宛先]

お世話になっております。
KicksWrapの[担当者名]です。

以下の商品の発注をお願いいたします。

【発注内容】
商品名：[商品名]
数量：[数量]個
希望納期：[納期]

[備考]

よろしくお願いいたします。
```

---

## 記入ガイド

| 項目 | 説明 |
|------|------|
| 宛先 | ヲヲフェニックスの担当者名 |
| 担当者名 | 発注を出す人の名前 |
| 商品名 | 正式名称 |
| 数量 | 発注者.mdのルールで算出された数 |
| 希望納期 | リードタイムを考慮した日付 |
| 備考 | アッセンブリ指示、急ぎの旨など |

※ このテンプレートは普段のやり取りに合わせて書き換えてOK
